INSTRUÇÕES DE USO:

1 - O programa segue o mesmo padrão dos arquivos do site 'http://morphett.info/turing/turing.html'.
2 - Deve ser usado Python versão 3.5.2, ou superior, para executar o programa, via linha de comando.
3 - Para exucutar o programa são necessários os seguintes parâmetros:
	3.1 - O usuario podera passar o tempo para execução continua
	3.2 - Ou o usuário não passando nada nos parametros terá uma execução continua com tempo de 0.05s
	3.3 - Ou podera passar o modo para uma execução pausada
	Exemplo: $ python3 turing_machine_structured.py s
4 - Após isso, será informado quais são os arquivos de configuração para escolha do usuario
	4.1 - Contudo, caso o usuário queira utilizar um arquivo proprio, antes da execução é necessário coloca-lo na 
	pasta files_config
5 - Após isso, será solicitado a palavra de entrada
6 - Caso o usuario tenha escolhido o modo pausado, a continuação se dá por meio da tecla Enter.
